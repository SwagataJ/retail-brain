# Data

All datasets and pipeline outputs. Raw CSVs are generated by `scripts/generate_data.py`.

## Raw Datasets

| File | Rows | Description |
|------|------|-------------|
| `customers.csv` | ~100,000 | Customer profiles with signup date and segment |
| `transactions.csv` | ~500,000 | Purchase transactions linked to customers and stores |
| `transaction_items.csv` | ~1,200,000 | Line items with unit price, quantity, and discount |
| `products.csv` | ~2,000 | Products across 5 categories with base/cost price and brand |
| `stores.csv` | ~100 | Store locations (mall, standalone, online) |
| `promotions.csv` | ~50 | Category and product-level promotions |
| `daily_sales.csv` | ~3,650 | Aggregated daily revenue by category (used for forecasting) |

## Output Subdirectories

| Directory | Source Script | Contents |
|-----------|--------------|----------|
| `forecasts/` | `predict_forecast.py` | 30-day forecast results and evaluation plot |
| `customer_intelligence/` | `customer_intelligence.py` | Segmentation, churn, affinity CSVs and visualization PNGs |
| `pricing_optimization/` | `pricing_optimization.py` | Elasticity, discount optimization, promotion timing CSVs and visualization PNGs |
